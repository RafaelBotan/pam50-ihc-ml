# Manuscript Files

This folder contains the public manuscript materials for the BCRT submission version.

- `manuscript_BCRT_v1.md`: editable manuscript source used to generate Word outputs.
- `manuscript_BCRT_v1.docx`: Word manuscript generated from the source.
- `Manuscript_BCRT_AllText_Tables_Figures.docx`: single editable peer-review file with text, tables, and main/supplementary figures placed in the body.
- `tables_BCRT.docx`: standalone table file generated for submission systems that request tables separately.

Administrative submission materials, cover letters, reviewer suggestions, generated package folders, and submission-system checklists are intentionally not tracked in this repository.
